Airline Reservation Database Project

This project simulates a complete airline reservation system using PostgreSQL. It includes relational modeling, schema creation, SQL views for analytics, and performance optimization features.

Key Features

  • 12+ normalized tables covering flights, bookings, airports, passengers, and staff
  • Views for Sales, Delays, and Flight Schedules
  • Queries for daily operations and performance monitoring
  • Performance considerations like indexing, caching, and partitioning
